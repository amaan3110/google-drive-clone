<h2>🛠️ Installation Steps:</h2>

<p>1. step 1</p>

```
git clone https://github.com/amaan3110/google-drive-clone.git
```

<p>2. step 2</p>

```
npm install
```

<p>3. step 3</p>

```
npm run dev
```

<p>4. step 4</p>

```
open localhost:5173 in your browser
```
